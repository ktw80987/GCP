[실습 1]
> 간단한 텍스트 기반 정보 표시 애플리케이션 만들기

<목표>
사용자는 간단한 텍스트 입력과 선택 요소를 통해 정보를 입력하고,
서버는 입력된 정보를 바탕으로 맞춤형 메시지를 표시하는 Streamlit 애플리케이션입니다.

<요구사항>

1) 이름을 입력받는 텍스트 입력 필드 추가
2) 나이를 입력받는 텍스트 입력 필드 추가
3) 성별을 선택할 수 있는 라디오 버튼 추가
4) 입력된 정보를 바탕으로 맞춤형 환영 메시지와 나이에 따른 간단한 메시지 표시

ex) ~~ 님 어서오세요. 미성년이시네요. 부모 동의가 필요합니다.

[실습 2]
> 간단한 데이터 시각화 애플리케이션 만들기

<목표>
사용자는 CSV 파일을 업로드하고,
서버는 업로드된 데이터를 데이터프레임으로 표시하며, 특정 열을 선택하여 해당 열의 히스토그램을 그리는 Streamlit 애플리케이션을 만듭니다.

<데이터>
data 폴더의 iris.csv 파일 사용하세요

<요구사항>
1) CSV 파일을 업로드할 수 있는 기능 구현
2) 업로드된 데이터를 데이터 프레임으로 표시
3) 데이터 프레임의 열 이름을 선택할 수 있는 드롭다운 메뉴 추가
4) 선택된 열의 히스토그램을 그리기
