1. 사이드바와 세션 상태를 활용한 간단한 웹 애플리케이션 만들기
<목표>
사용자는 사이드바를 통해 정보를 입력하고,
서버는 세션 상태를 활용하여 입력된 정보를 유지 및 갱신하는 Streamlit 애플리케이션을 만듭니다.

<요구사항>

1) 사이드바에 이름과 나이를 입력받는 텍스트 입력 필드 추가
2) 메인 화면에 현재 입력된 이름과 나이를 표시
3) 버튼을 클릭하면 이름과 나이를 리셋하여 기본값으로 되돌리기
4) 세션 상태를 이용하여 입력된 정보가 유지되도록 구현

2. 콜백 기능을 활용한 웹 애플리케이션 만들기
<목표>
사용자는 사이드바를 통해 정보를 입력하고,
서버는 세션 상태와 콜백 기능을 활용하여 버튼 클릭 시 정보를 업데이트하거나 리셋하는 Streamlit 애플리케이션을 만듭니다.

<요구사항>

1) 사이드바에 이름과 나이를 입력받는 텍스트 입력 필드 추가
2) 메인 화면에 현재 입력된 이름과 나이를 표시
3) 'Update' 버튼을 클릭하면 입력된 이름과 나이를 세션 상태에 업데이트
4) 'Reset' 버튼을 클릭하면 이름과 나이를 기본값으로 되돌리기
5) 콜백 함수를 사용하여 버튼 클릭 시 동작을 처리

- 콜백함수명 : update_info(), reset_info()
